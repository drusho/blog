About the Data

Web scraping was performed on the Top 10 Tech Channels on Youtube using Selenium (an automated browser (driver) controlled using python, which is often used in web scraping and web testing). The youtube channels to were scrapped were determined using a Top 10 Tech Youtubers list from blog.bit.ai. Scraping included:

  • Channel names
  • Number of subscribers per channel
  • Data from most popular videos per channel included:
    • Video titles
    • Posting date
    • Number of views

The average number of videos per channel was around 200. In total, the data from 2000 videos was scrapped.
